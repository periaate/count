# count
`count` counts the number of arguments received and prints the count. Count works with both piped inputs and arguments.

## Usage
`ls | count`\
`count (ls)`\
`count arg1`\
`ls | count arg1`\
`ls | count (ls)`\
`ls | count arg1 (ls)`
